# Asteroids

A classic Asteroids clone built with Python and pygame.

## How it works
You control a spaceship in a 2D space field. Asteroids of various sizes spawn at the edges of the screen and drift towards the center. You can move the ship, rotate, and shoot the asteroids. 
When a large asteroid is hit, it splits into two smaller, faster-moving asteroids. When a small asteroid is hit, it is destroyed completely.
Survive as long as possible without crashing into an asteroid!

## Instructions

### Requirements
- Python 3.13+
- pygame

### Running the game
1. Install dependencies:
   ```bash
   pip install pygame
   ```
2. Run the game from your terminal:
   ```bash
   python main.py
   ```

### Controls
- **W** - Move forward
- **S** - Move backward
- **A** - Rotate left
- **D** - Rotate right
- **SPACE** - Shoot
